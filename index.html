❓ How it works

🚀 Fully Automatic!
Drop an image → it uploads → search engines open with your image pre-loaded.

How the magic happens:

  1. You drop/paste an image
  2. Image auto-uploads to Catbox (temp 1hr host)
  3. Public URL generated
  4. Search engines open with the URL — image pre-loaded!

For URL images:

If you paste a URL, no upload needed — we pass the URL directly to search engines.

Privacy note:

Uploaded images are stored on litterbox.catbox.moe temporarily (1 hour) then auto-deleted. This is necessary so Google/Yandex/etc can fetch your image.

Search engines:

  • Google Lens — Best for general recognition
  • Yandex — Great for finding higher res versions
  • Bing — Good for products
  • TinEye — Find where image appears online
  • SauceNAO — Best for anime/artwork

⚠️ Popup blocker:

If only one tab opens, your browser is blocking popups. Click the blocked popup icon in your address bar and select "Always allow popups from this site". Then click "Search Again".
